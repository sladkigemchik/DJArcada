# Игровая платформа Arcada

## Обзор

Этот проект представляет собой динамичную игровую платформу, которая позволяет пользователям находить, делиться и даже модерировать контент о различных играх. В нем решается задача создания интерактивного центра сообщества, где игроки могут публиковать статьи, классифицировать их и даже взаимодействовать с ИИ по вопросам, связанным с играми, обеспечивая при этом качество контента с помощью инструментов модерации.

## Описание

Добро пожаловать на игровую платформу Arcada — динамичное веб-приложение, созданное с использованием Django, которое по-новому определяет способы взаимодействия геймеров с игровым контентом и их вклад в его создание. Оно предназначено для поиска статей об играх, содействия созданию контента, управляемого сообществом, и предоставления мощных инструментов модерации для поддержания качества. От подробных статей об играх и интуитивно понятной классификации до встроенного чата с искусственным интеллектом и надежного управления пользователями — Arcada предлагает широкий спектр услуг как для обычных игроков, так и для преданных своему делу энтузиастов игр.

## Установка

Запустить этот проект локально довольно просто:

1. **Клонируйте репозиторий**:
    ```bash
    git clone https://github.com/sladkigemchik/DJArcada.git
    cd DJArcada
Настройте виртуальное окружение:
Рекомендуется использовать виртуальное окружение для управления зависимостями.

bash
python -m venv venv
Активируйте его:

На Linux/macOS:

bash
source venv/bin/activate
На Windows:

bash
.\venv\Scripts\activate
Установите зависимости:
Все необходимые пакеты Python перечислены в requirements.txt.

bash
pip install -r requirements.txt
Примените миграции базы данных:
Это настроит схему вашей базы данных SQLite.

bash
python manage.py makemigrations arcada users
python manage.py migrate
Создайте суперпользователя (учетную запись администратора):
Вам понадобится учетная запись администратора для доступа к панели администратора Django и управления контентом/пользователями.

bash
python manage.py createsuperuser
Следуйте инструкциям, чтобы ввести свое имя пользователя, адрес электронной почты и пароль.

Запустите сервер разработки:

bash
python manage.py runserver
Теперь откройте ваш веб-браузер и перейдите по адресу http://127.0.0.1:8000/.
